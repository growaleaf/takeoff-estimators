About these puzzles. Every excerpt is an illustrative teaching example — composed in standard CSI MasterFormat section format, not lifted from any real project's proprietary specifications. What is real is the conflict logic: each trap mirrors a documented pre-bid failure mode — the Division 23↔26 "who powers the damper" seam, owner-furnished connection responsibility, "Not Used" sections that are still cross-referenced, allowance/base double-coverage, supplementary-condition overrides, and so on. Section numbers, defined terms (Furnish / Install / Provide), and standards references (NFPA, ASSE, NETA, ASCE 7, UL firestop systems) reflect real conventions. Nothing here asserts what any specific jurisdiction currently enforces — code-edition puzzles keep the conflict internal to the documents shown.
