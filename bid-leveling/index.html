How to read it like an estimator

  1. Base price is a headline, not a total. An exclusion pushes scope onto someone else — but the project still buys it.
  2. Only asymmetric, contractor-owned gaps move the ranking. If every bid excludes the same thing, or the owner furnishes it (OFCI), it's a wash.
  3. Size the gap before you flip. A $2.5k access-panel exclusion doesn't beat a $55k base spread; a $165k controls exclusion does.
  4. Level to a common scope, then compare. Lowest leveled total wins — that's the true low.
